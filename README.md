## PFO - 1

### Consigna:

Aprender a configurar un servidor de sockets en Python que reciba mensajes de clientes,
los almacene en una base de datos y envíe confirmaciones, aplicando buenas prácticas de
modularización y manejo de errores.

## Ejecución

1. Clona el repositorio:

   ```bash
   git clone https://github.com/Ferclemens/PR_3A1C26_PFO1_Clemens_Fernando.git
   cd PR_3A1C26_PFO1_Clemens_Fernando
   ```

2. Iniciar servidor:
   ```bash
    python servidor.py
   ```
   Abrimos otra terminal e iniciamos el cliente

3. Iniciar cliente:
   ```bash
   python cliente.py
   ```
4. Escribimos los mensajes que queramos desde la terminal del cliente. Par verlos reflejados en el servidor. Salimos con el mensaje "éxito" (sin comillas).
